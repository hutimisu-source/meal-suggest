# meal-suggest

# 献立提案プログラム

## 概要
入力された食材をもとに、作成可能な料理を提案するC言語プログラムです。

## 開発目的
日々の献立作成に時間を要している課題に着目し、その解決を目的に開発しました。

## 使用技術
- C言語

## 機能
- 食材の選択入力
- 条件一致による料理検索
- 結果表示（調理時間・URL付き）

## 工夫点
- 構造体を用いたデータ管理
- 食材一致判定ロジックの実装
- エラーコードによる例外処理

## 実行方法
Visual Studio 2022で実行可能

## 今後の改善
- Webアプリ化
- 献立データの拡充
